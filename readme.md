# Bilibili Remuxer

A remuxer for BiliBili Android Application
(tv.danmaku.bilibili, tv.danmaku.bilibilixl, com.bilibili.app.blue).

## Dependencies

1.Python(>=3.4)

2.FFmpeg(Compiled)

## Usage
1. Filling FFmpeg's path in config.py.

2. Run it.
```
python remuxer.py [sdcard path with \ or /]
```
